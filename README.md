# TaskFlow 📝

TaskFlow is a modern, lightweight Android task manager designed to help you organize your daily life with ease. Built with focus on clean UI and smooth user experience.

## ✨ Features

- **Effortless Task Management**: Add, edit, and delete tasks with a simple interface.
- **Priority Levels**: Color-coded priorities (High 🔴, Medium 🟠, Low 🟢) to help you focus on what matters.
- **Dynamic Themes**: Full support for Light and Dark modes with a custom-designed toggle.
- **Data Persistence**: Your tasks are saved automatically and persist even after app restarts.
- **Intuitive UI**: 
    - Swipe-to-delete functionality.
    - Long-press to edit existing tasks.
    - Automatic sorting (Incomplete tasks and High priority stay at the top).
- **Modern Design**: Material Design components with a custom branded splash screen.

## 🛠️ Built With

- **Java**: Core programming language.
- **RecyclerView**: For smooth list handling.
- **SharedPreferences & JSON**: For lightweight data storage.
- **Material Design**: For a professional look and feel.

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/TaskFlow.git
   ```
2. Open the project in **Android Studio**.
3. Build and run on an emulator or physical device.
